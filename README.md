# Spice Website

This is a static website project showcasing various spices, including cardamom, cinnamon, cumin, ginger, pepper, and turmeric. It features informational pages for each spice, along with a gallery and contact page.

## Project Structure

- **Types**: HTML, CSS, JavaScript
- **Pages**:
  - `home.html`: Main landing page
  - `idukki.html`: Information about Idukki
  - Individual spice pages (e.g., `pepper.html`, `ginger.html`)
  - `gallery.html`: Photo gallery
  - `contactus.html`: Contact form
- **Assets**: Located in the `asset` directory (css, js, image)

## Usage

To view the website, open `home.html` or any other `.html` file in your preferred web browser.

## Note on Images

Some images used in this project are missing. You can add your own images to the `asset/image` directory to complete the visual experience.
